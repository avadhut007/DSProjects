Notes before running files
-------------------------------
- Each time you close client you should close server as well.

- Before starting the client, restart the server always.

- Python3 version > 3.7  required

- Install numpy library using pip
pip3 install numpy

- if you pressed enter instead of required value, you may need to restart the whole process.

Steps:
-------------------------------

1. Start two terminals and change directory to Assignment-4

2. Run server.py in the first terminal.
python3 server.py

3. Run client.py in the second terminal.
python3 client.py

4. Select options from list of instructions on the terminal. To select type the number of option and enter.

5. In the end, close the client and close the server as well.
